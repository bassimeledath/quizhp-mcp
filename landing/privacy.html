Privacy Policy

Last updated: March 18, 2026

This Privacy Policy explains how Hunch AI, LLC ("Hunch AI," "QuizHP," "we," "us," or "our") handles your information when you use the QuizHP MCP server and related services (the "Service").

By using the Service, you acknowledge that you have read and understood this Privacy Policy.

1. What QuizHP Does

QuizHP is an MCP (Model Context Protocol) server that generates interactive quiz games. When you ask an AI assistant (Claude or ChatGPT) to create a quiz, the assistant generates questions and sends them to QuizHP, which returns an interactive game widget. No account or authentication is required.

2. Information We Collect

QuizHP collects no personal data. Specifically:

  • No user accounts or authentication required
  • No cookies or tracking
  • No analytics or telemetry
  • No conversation data stored
  • No IP addresses logged
  • No payment or financial information

3. Data We Process Temporarily

When a quiz is created, the questions and answers are held in server memory for up to 1 hour to support the active game session. This data is:

  • Not written to any database or persistent storage
  • Not shared with any third party
  • Automatically deleted when the session expires or the server restarts

4. How We Use Information

The only information processed is quiz question data sent by the AI assistant. This data is used solely to:

  • Render interactive quiz game widgets
  • Track game session state (correct/incorrect answers) during the active session

5. How We Share Information

We do not sell personal information. The only third party involved in processing is:

  • Cloudflare (hosting) — Cloudflare may process connection metadata (IP addresses, TLS info) per their own privacy policy. QuizHP itself does not access or store this metadata.

6. Data Retention

  • Quiz session data is held in memory for up to 1 hour, then automatically deleted
  • No data is persisted to any database or file storage
  • Server restarts clear all session data

7. Your Rights

Since QuizHP collects no personal data, there is no personal data to access, correct, or delete. If you have questions about data handling, contact us at the address below.

8. GDPR / UK-GDPR

If you are located in the EU or UK: QuizHP processes no personal data as defined under GDPR. No data controller registration is required. Connection metadata processed by Cloudflare is governed by Cloudflare's privacy policy and Data Processing Addendum.

9. CCPA / California Privacy Rights

If you reside in California: QuizHP does not collect, sell, or share personal information. No categories of personal information are collected.

10. Children's Privacy

QuizHP is a general-audience service. We do not knowingly collect any personal information from anyone, including children under 13. Since no personal data is collected, there is no risk of inadvertent collection.

11. Data Security

We implement reasonable security measures including:

  • Encryption in transit (HTTPS/TLS)
  • Stateless architecture with no persistent data storage
  • Cloudflare's infrastructure security

12. Changes to This Policy

We may update this Privacy Policy from time to time. Changes will be posted at this URL. Your continued use of the Service signifies acceptance of the updated policy.
