How it works

Three steps to interactive quizzes

Connect the MCP server, ask for a quiz, and play. Each question becomes a unique mini-game.

1

Connect

Add the MCP server URL to Claude Desktop, Claude.ai, or ChatGPT. Takes 30 seconds, no API keys needed.

2

Ask for a quiz

Tell the AI what you want to be quizzed on. Any topic works — science, history, your own documents, code concepts.

3

Play interactive games

Each question renders as a different mini-game — archery, treasure chests, switches, potion bottles, and 120+ more.

Install

Get started in 30 seconds

No API keys, no accounts. Just add the server URL and start quizzing.

Claude Desktop & Claude.ai

Add this to your claude_desktop_config.json:

{
  "mcpServers": {
    "quizhp": {
      "url": "https://mcp.quizhp.com/mcp"
    }
  }
}

Works with Claude Desktop, Claude.ai, and Claude Code (via MCP settings).

ChatGPT

Connect the MCP server through ChatGPT's settings:

  1. Open Settings in ChatGPT
  2. Go to Connected Apps
  3. Click Add MCP Server
  4. Enter the server URL:
https://mcp.quizhp.com/mcp

Then ask ChatGPT to quiz you on any topic. It will use QuizHP automatically.
